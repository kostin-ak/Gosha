<h1>ГОША</h1>

ИИ, подходящий для выполнения простейших функций, обладающий возможностью практически не затрудненного разговора. 

Для использования требуются модули:

  argparse

  newsapi-python

  wikipedia-api

  vk_api

  pandas

  pygame

  pyaudio

  pywin32

  pypiwin32

  speechrecognition

  pyttsx3

  Plyer

  bs4

  urllib

  PyOWM 


Которые можно установить благодаря командам:

  pip install argparse

  pip install newsapi-python

  pip install wikipedia-api

  pip install vk_api

  pip install pandas

  pip install pygame

  pip install pyaudio

  pip install pywin32

  pip install pypiwin32

  pip install speechrecognition

  pip install pyttsx3

  pip install Plyer

  pip install bs4

  pip install urllib

  pip install PyOWM


Или воспользовавшись .bat файлом с именем modul в папке /bin.

PS: Полно багов и личных строк, а также проект перый(из ии), посему не ругайтесь.

PPS: Ключи запуска:

<code>                -dbg(Debug)</code>
